Question 3
Load the Gross Domestic Product data for the 190 ranked countries in this data set: 

https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FGDP.csv 

Load the educational data from this data set: 

https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FEDSTATS_Country.csv 

Match the data based on the country shortcode. How many of the IDs match? Sort the data frame in descending order by GDP rank (so United States is last). What is the 13th country in the resulting data frame? 

gdpurl = "https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FGDP.csv"
eduurl = "https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FEDSTATS_Country.csv"
q3gdp = read.csv(gdpurl)
q3edu = read.csv(eduurl)
q3gdp <- q3gdp[c(5:194),]
colnames(q3gdp)[c(1:3)] <- c("CountryCode","GDP","NA")
q3merge <- merge(q3edu, q3gdp, by="CountryCode")

> q3merge <- merge(q3gdp, q3edu,all = FALSE)
> View(`q3merge`)
> q3merge$GDP <- as.numeric(as.character(q3merge$GDP))
> q3merge[order(q3merge$GDP,decreasing = TRUE),1][13]
